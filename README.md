# AstrBot llss_sub 插件

一个根据用于订阅琉璃神社更新的AstrBot插件，定时爬取最新文章并推送到群聊。
请合理设置刷新间隔和最大爬取页数，避免浪费网站资源，多多点赞、评论支持神社的更新。

## 🛠️ 安装步骤

1. **克隆插件到AstrBot插件目录**
```bash
cd /path/to/astrbot/plugins
git clone https://github.com/Joker42S/astrbot_plugin_llss_sub.git
```

2. **重启AstrBot**
 
3. **配置参数**
   - 配置可用的神社地址，或使用默认地址


## 📖 使用方法

```bash
# 在群聊中发送，为该群聊订阅更新
/订阅神社

# 立即检查并推送一次更新
/刷新神社
